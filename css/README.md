LUMINA

Interactive learning platform with flashcards, quizzes, performance tracking, and gamification. 

 Tech Stack:
- Frontend:React (responsive, mobile-first, PWA-ready)
- Backend:Node.js + Express
- Database:*ongoDB
- Auth: JWT, password hashing
- API: RESTful, clean frontend-backend separation
 Features:
- Flashcards:Definitions, theorems, formulas; topic/chapter-wise; swipe animations
- Document Upload: PDF/DOC
- Quiz Engine: Easy, Medium, Hard; track right/wrong answers
- Performance Analytics:Recommend flashcards, adjust quiz difficulty
- Progress Tracking:Charts, topic-wise completion, learning trends
- Gamification:Streaks, points, badges, achievements
- Group Study: Chat, collaborative quizzes, shared dashboards
- User Dashboard: Streak, progress summary, weak/strong topics, history

Deliverables

- Frontend & backend code
- Database schema & API routes
- Sample seed data
- Clear folder structure & setup instructions
- Well-commented, production-ready code
